## 🧩 Description

A Qt-based circuit design and simulation environment that combines the Qucs GUI with SPICE-compatible backends such as Ngspice, Xyce, SpiceOpus, and QucsatorRF.

---

## 🚀 Key Features

<!-- FEATURES:START -->
- Unified GUI for multiple simulation kernels including Ngspice, Xyce, SpiceOpus, and QucsatorRF
- Supports AC, DC, transient, S-parameter, FFT, distortion, pole-zero, and parametric sweep analyses
- Direct integration of SPICE models from component datasheets
- Quick switching between simulation backends without restarting the application
- Includes RF-oriented capabilities such as QucsatorRF integration and tuner simulation mode
<!-- FEATURES:END -->
